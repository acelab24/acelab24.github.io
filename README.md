# ACE Lab Homepage

강원대학교 응용·계산 전자기학 연구실(Applied & Computational Electromagnetics Lab) 홈페이지.

- **라이브 사이트**: GitHub Pages (Settings → Pages에서 `main` 브랜치 활성화)
- **구조**: `index.html` 단일 파일 (Tailwind CDN + Pretendard + Lucide icons)
- **수정 방법**: `index.html`을 직접 편집 후 commit & push — 1~2분 내 자동 반영
- **연락처**: cspark@kangwon.ac.kr
